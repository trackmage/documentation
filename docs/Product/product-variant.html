ProductVariant

team
REQUIRED

Type: string (format: iri-reference)

The team reference to which the product variant belongs.

Example: /shipments/83829499-f99c-4c91-9908-76c91f2ae312

sku
REQUIRED

Type: string|null

The SKU of product variant.

Example: JACKET-BLACK

imageUrl

Type: string|null (format: URL)

The URL of product variant image in the ecommerce store.

Example: http://my-store/media/products/variants/jacket-black.jpg

product

Type: string (format: iri-reference)

The product reference to which the product variant belongs.

Example: /products/625c21dc-1e78-4b5c-8e29-5a1d7f84f4ac

price

Type: float|null

The price of product variant in the store currency. Default value is 0

Example: 9.99

externalSourceIntegration

Type: string|null (format: iri-reference)

The workflow reference to integration for ecommerce store.

Example: /workflows/a0b97939-5aea-4b35-b7c6-64f56bff1721

externalSourceSyncId

Type: string|null

The id of the shipment in ecommerce store (WooCommerce, Shopify, etc.).

Example: wp_00001

id
READ-ONLY

Type: string (format: uuid)

A unique version 4 UUID.

Example: 601ccee6-d07a-4d0e-a376-65d73f1f182a
