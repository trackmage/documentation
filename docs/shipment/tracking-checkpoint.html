Tracking Checkpoint

Tracking Checkpoint Properties#

shipment

Type: string (format: iri-reference)

The shipment reference to which the shipment item belongs.

Example: /shipments/83829499-f99c-4c91-9908-76c91f2ae312

checkpointDate

Type: string (format: date-time)

The date and time (in ISO-8601 format) of the checkpoint, provided by carrier.

Example: 1608193517

info

Type: string|null

Checkpoint message provided by carrier.

Example: We deliver your shipment at your preferred day and/or time

location

Type: string|null

Location info provided by carrier.

Example: BOSTON MA DISTRIBUTION CENTER

rel

Type: string|null

carrier affiliation

Example: origin

id
READ-ONLY

Type: string (format: uuid)

A unique version 4 UUID.

Example: 2f7fabde-f99d-4160-a90a-20caa2fb95e6
