Carriers

What you can do with Carrier#

  • GET /public/carriers
    Retrieves the collection of Carrier resources.
  • GET /public/carriers/{id}
    Retrieves a Carrier resource.

Carrier Properties#

code

Type: string|null

The carrier code. Used as identifier

Example: china-post

name

Type: string|null

The carrier name

Example: China Post

phone

Type: string|null

The carrier phone

Example: +861011185

url

Type: string|null

The carrier website url

Example: http://english.chinapost.com.cn/

image

Type: string|null

The carrier logo url

countryIso2Codes

Type: array

ISO2 codes of countries this carrier belongs to

Example: CN

description

Type: string|null

The carrier description

Test tracking numbers#

These tracking numbers will be detected as TrackMage Test carrier and have pre-defined status. Use them for the testing purposes only.

Each tracking number consists of the following mask: TM###XXXXXXXXXXXXTM, where X can be changed to any value, for example TM2020002TM or TM20299999TM will have Available for Pickup status.
