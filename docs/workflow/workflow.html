Workflows

What you can do with Workflow#

  • GET /workflows
    Retrieves the collection of Workflow resources.
  • GET /workflows/{id}
    Retrieves a Workflow resource.
  • GET /workflows/{id}/runs
    Retrieves the collection of WorkflowRun resources.
  • GET /workspaces/{id}/workflows
    Retrieves the collection of Workflow resources.
  • GET /workspaces/{id}/workflows/{workflows}/runs
    Retrieves the collection of WorkflowRun resources.
  • GET /teams/{id}/workspaces/{workspaces}/workflows
    Retrieves the collection of Workflow resources.
  • GET /teams/{id}/workspaces/{workspaces}/workflows/{workflows}/runs
    Retrieves the collection of WorkflowRun resources.
  • POST /workflows
    Creates a Workflow resource.
  • POST /workflows/execute
    Creates a Workflow resource.
  • POST /workflows/submit-suggestion
    Creates a Workflow resource.
  • PUT /workflows/{id}
    Replaces the Workflow resource.
  • DELETE /workflows/{id}
    Removes the Workflow resource.
